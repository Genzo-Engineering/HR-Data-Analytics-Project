# 📊 Projet : Optimisation RH par l'Analyse de Données

## 📝 Introduction
Suite à mes échanges avec des Data Engineers, j'ai priorisé la partie **Reporting** pour démontrer la valeur métier de la donnée. Ce projet va du nettoyage SQL à la visualisation stratégique.

## 🛠️ Stack Technique
- **Base de données :** SQL Server (SSMS)
- **Traitement :** T-SQL / Python (Jupyter Notebook)
- **Visualisation :** Power BI

## 🧹 Ce que j'ai fait
1. **Nettoyage SQL :** Normalisation des dates, gestion des erreurs de saisie et calcul de l'âge.
2. **Analyse métier :** Réponse à 11 questions clés (Turnover, diversité, répartition géographique).
3. **Dashboard :** Création d'un rapport interactif pour les décideurs RH.
